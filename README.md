## üß™ Installation & Usage

1. Clone the repository to your local machine:

```bash
git clone https://github.com/Gautierpicon/iPhone15Pro-website.git
```

2. Go to the project directory:

```bash
  cd iPhone15-website
```

3. Install dependencies:

```bash
  npm install
```

4. Start the server:

```bash
  npm run dev
```

5. Visit:

```bash
http://localhost:5173/
```
